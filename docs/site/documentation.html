Installation Guides

ComPAIR requires a minimum of a web server running python 2.7 or 3 and a mysql or mariadb database. It is recommended to also have Redis and a Celery daemon running to reduce server load for certain tasks.

  • CentOS 7 Server (Standalone Linux): Setup an all-in-one ComPAIR server. Useful for piloting ComPAIR or small scale production environments.
  • Amazon Web Services (AWS): Setup ComPAIR on Amazon Web Services. Useful for larger scale production environments.
  • Google Container Engine / Kubernetes: Setup ComPAIR on Google Container Engine or Kubernetes. Useful for larger scale production environments.
  • Development Instance via Docker: Setup a development environment for ComPAIR. Useful for contributing to ComPAIR or customizing code. Not recommended for use in a production environment. Instead follow the Kubernetes guide above.
