Import Users

Create, enrol, or drop multiple users in this course at the same time by importing a CSV file with a full list of your course's users.

Requirements

Note the following:

  • This application only supports importing a file that uses the CSV (comma-separated values) format.
  • User information is expected in this order: Username, Student Number, First Name, Last Name, Email, Display Name, Password (examples below).
  • Only usernames are required. All the other fields are optional. If a display name is not given, the user is assigned an anonymous display name default, e.g., "Student1234". If a password is not given, the default password is the username. All other empty fields are left blank.
  • Usernames and student numbers must be unique in the application. Duplicates will be altered on import automatically.
  • Each new upload should include the full list of users you want in the course. Any new users in an imported file will be enrolled in the course and any deleted users will be dropped from the course.
