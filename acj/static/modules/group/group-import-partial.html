Assign Groups

Assign users who are already enrolled in your course to specific groups in the course.

Note the following guidelines:

  • Groups are assigned by importing a CSV (comma-separated values) file.
  • Data is expected in this order: User Identifier, Group Name. Both fields are required.
  • The user identifier can be the username in the application or the student number.
  • To export a list of student numbers to use for a CSV file, UBC instructors can log in to the Faculty Service Centre.
