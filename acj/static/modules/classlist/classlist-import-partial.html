Import Users


  • This application supports importing a file that uses the CSV (comma-separated values) format.
  • Data is expected in this order: Username,Student Number,First Name,Last Name,Email,Display Name,Password (examples below).
  • Only usernames are required. All the other fields are optional.
  • Usernames, student numbers, and display names must be unique in the application. Duplicates will be altered by the application automatically.
  • Each upload should include the full list of users you want in the course. To remove or add students, simply upload a CSV file with an updated list.

Formatting examples

The data examples below show how to drop non-required fields.

No Display Name or Password:
student1,,Jack,Smith,jack.smith@student.ca
No Display Name:
student1,,Jack,Smith,jack.smith@student.ca,,password123
No First Name or Last Name:
student1,,,,jack.smith@student.ca,display0001, password123

Note that:

  • If a display name is not given, the user is assigned an annonymous display name default, e.g., "Student1234".
  • If a password is not given, the default password saved for the user is the username.
