Upload Enrolment - {{course.name}}

Format (CSV)

Username,First Name,Last Name,Email,Display Name,Password

  • Only usernames are required. All the other fields are optional.
  • If a password is not given, the password for the user is their username.

Example

  • student1,Jack,Smith,jack.smith@student.ca
    • No Display Name and Password
  • student1,Jack,Smith,jack.smith@student.ca,,password123
    • No Display Name
  • student1,Jack,Smith,jack.smith@student.ca,display0001
    • No Password
  • student1,Jack,Smith,jack.smith@student.ca,display0001,password123
