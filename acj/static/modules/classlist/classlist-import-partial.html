Import Users for {{course.name}}


  • This application supports importing files that use the CSV (comma-separated values) format.
  • Data is expected in this order: Username,First Name,Last Name,Email,Display Name,Password
  • Only usernames are required. All the other fields are optional.

Formatting examples

The data examples below show how to drop non-required fields. If a password is not given, the password saved for the user is the username.

No Display Name or Password:
student1,Jack,Smith,jack.smith@student.ca
No Display Name:
student1,Jack,Smith,jack.smith@student.ca,,password123
No First Name or Last Name:
student1,,,jack.smith@student.ca,display0001, password123
